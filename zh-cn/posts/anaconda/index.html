今天运行 anaconda 突然无法打开启动界面，但可以直接打开并运行 jupyter 和 notebook。报错情况如下图所示。

网络上没有和我相同的情况，最后我是打开anaconda powershell prompt 看到了具体问题，是路径识别出现问题。

原因是 Windows 环境变量冲突：我的用户变量 path 和环境变量 path 设置冲突，不一致。导致在用户变量的 anaconda 被识别到了系统变量的 Java 环境。

解决方法：统一用户变量和系统变量、或者删除 java 路径。这里个人是因为无效的 path 路径挡路了。个人删除了 Java 的错误路径：%Java\_Home%\bin;%Java\_Home%\jre\bin。之后检验 Java 环境没受到影响，anaconda 环境也恢复了。
